# Custom Agent - Study Assistant

## Implementation:
A custom AI agent was created to act as a study assistant. 
The agent uses multiple tools, including a calculator for mathematical and a file-reading tool to access local documents.
The agent is designed to help a student to understand course material and answer questions in a clear and strutured way.

## Result: 
The agent can solve calculations, read local files, and explain concepts such as MLOps.
This demonstrate how multiple tools can be combined in a single agent to solve practical tasks and support learning.
