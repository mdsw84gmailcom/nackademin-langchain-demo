# Experiment top_p temperature

## Test 1 
Settings:
temperature = 0.1
top_p = 0.5

## Results:
The response from the model was relatively stable and fact-based. 
It provided a clear defination of what an AI agent is and listed several examples of functions and uses. 
The response was structured and focused on basic information.

## Test 2
temperature = 0.9
top_p = 0.95

## Reultat:
The response was more varied and somewhat more creative. 
The model provided more types of examples and formulations. 
The response also contained a more simplified description of how an AI agent works and what parts it might consist of.

Reflektion:
When the temperature is low, the model's answers are more stable and fact-based. 
When the temperature is high, the answers are more varied and creative.

Top-p affects how large a selection of possible words the model can choose from. 
A higher value gives the model more possible formulations and can therefore create more varied answers.
