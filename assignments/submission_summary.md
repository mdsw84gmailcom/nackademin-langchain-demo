# Submission Summary - Three AI Agents

This submission contains three AI agents created from the provided LangChain project template.

1. Memory Agent
Purpose: To maintain conversational context across multiple interactions.
Functionality: Stores previous messages and uses conversation history when answering.

2. Tool Agent
Purpose: To solve mathematical pproblems accurately.
Functionality: Uses a calculator tool to evaluate expression.

3. File-Reading Agent
Purpose: To answer questions based on local file content.
Functionality: Uses a file-reading tool to access and analyze a local text file.

Each agent includes a structured system prompt defining role, purpose, behavior, language, and constraints.
