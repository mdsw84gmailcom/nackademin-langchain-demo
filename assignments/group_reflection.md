# Group Reflektion - AI Agents

1. Demo (Purpose and Implemantation):
The purpose of the agent is to assist a student by answering, reading files, and performing calculations.
The agent was implemented using LangChain with tool such as calculator and file reader reader.

2. Use in organization:
Such an agent could be used as an internal assistant in a company.
For example, it could help employes find information in documents, answer questions about internal processes, or assist with basic data analysis.

3. Risk and Mitigation: 
There are several risks when using AI agents:
- Incorrect of misleading answers
- Security risks when accessing files or data
- Over-reliance on AI

These risks can be reduced by:
- Limiting access to sensitive data
- Logging and monitoring usage
- Adding human review for critical tasks

4. Selected Agent (Presentation summary)
The selected  agent is a study assistant agent.
It helps users understand information, read documents, and perform calculations.

The main risk is that the agent might provide incorrect explanations, so it should be used as a support tool rather than a single source of truth.
